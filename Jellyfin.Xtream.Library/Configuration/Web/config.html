Xtream Library has migrated to a new plugin GUID. This entry will not receive any further updates.

Follow the three steps below to move to the new plugin. Your existing setup keeps working in the meantime.

  1. Open the plugin catalog and search for Xtream Library. You will see a separate entry alongside this one (different GUID, different icon).
  2. Install the new entry. Restart Jellyfin when prompted. You now have two "Xtream Library" plugins installed for a moment — that is expected.
  3. Come back to this page, then uninstall this plugin (the old one) and restart Jellyfin one more time.
Your settings carry over automatically. Provider credentials, library folders, channel selections, and sync schedule live in a config file keyed by plugin namespace, not GUID — the new install picks them up on first start. No reconfiguration needed.
